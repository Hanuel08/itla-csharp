# Curso de C# - ITLA

Este repositorio contiene las prácticas correspondientes al curso de **Programación C#** en el **Instituto Tecnológico de Las Américas (ITLA)**.

## 🚀 Estructura del Proyecto

El proyecto está organizado en diferentes carpetas de prácticas, cada una enfocada en conceptos clave del lenguaje C#:

*   **[practice-1](file:///home/hanuel/RiderProjects/itla/practice-1)**: Fundamentos básicos.
    *   Conversor de grados (Celsius/Fahrenheit).
    *   Verificador de palíndromos.
    *   Operaciones aritméticas y lógica elemental.
*   **[practice-2](file:///home/hanuel/RiderProjects/itla/practice-2)**: Manipulación de cadenas y fechas.
    *   Métodos y validación de strings.
    *   Concatenación y formato.
    *   Trabajo con números y tipos de fechas (`DateTime`).
*   **[practice-3](file:///home/hanuel/RiderProjects/itla/practice-3)**: Estructuras de control y lógica comercial.
    *   Menús interactivos y calculadoras.
    *   Control de acceso y gestión de usuarios.
    *   Cálculo de calificaciones, edades y lógica de tienda/compras.
*   **[practice-4](file:///home/hanuel/RiderProjects/itla/practice-4)**: Bucles y arreglos.
    *   Trabajo con matrices multidimensionales.
    *   Iteraciones y manipulación de arreglos.

---

## 🛠️ Requisitos y Ejecución

Para compilar y ejecutar los proyectos, necesitas tener instalado el **.NET SDK** (versión 10.0 o superior).

### Compilar la solución entera
Desde la raíz del repositorio, ejecuta:
```bash
dotnet build practice-1/practices.sln
```

### Ejecutar una práctica individual
Entra en la carpeta de la práctica deseada y ejecuta:
```bash
cd practice-1
dotnet run
```
*(Repite el proceso para `practice-2`, `practice-3` o `practice-4` según corresponda).*
